Ejercicios resueltos

Ejercicio 1

1. Escriba un programa para calcular la suma de los dos valores enteros dados. Si los dos valores son iguales, devuelve el triple de su suma.

Ejercicio 2

2. Escriba un programa para obtener la diferencia absoluta entre n y 51. Si n es mayor que 51, devuelva el triple de la diferencia absoluta.

Ejercicio 3

3. Escriba un programa para verificar dos enteros dados y devuelva verdadero si uno de ellos es 30 o si su suma es 30.

Ejercicio 4

4. Escriba un programa para verificar un entero dado y devuelva verdadero si es 10, 100 o 200.

Ejercicio 5

5. Escriba un programa para verificar si un número positivo dado es un múltiplo de 3 o un múltiplo de 7.

Ejercicio 6

6. Escriba un programa para verificar dos números enteros dados si alguno de ellos está en el rango 100..200 inclusive.

Ejercicio 8

8. Escriba un programa para verificar si tres valores enteros dados están en el rango 20..50 inclusive.

Ejercicio 9

9. Escriba un programa para verificar si dos valores enteros dados están en el rango de 20..50 inclusive. Devuelve verdadero si 1 u otro está en dicho rango; de lo contrario, es falso.

Ejercicios con estructuras de datos

- Crear un array con los siguientes valores: 1,3,"a",[1,2].
- Pedir al usuario números hasta que ingrese la letra q. guardar los números en un array.
- Mostrar los números ingresados por el usuario multiplicados por 2.
- Mostrar los elementos del array que son primos.
- Devolver si existe un elemento dentro del array.
- Filtrar todos los elementos que no sean de tipo numerico.
- Ordenar los elementos de menor a mayor.

Ejercicios con diccionarios

Dada la siguiente estructura:

{
"nombre": "pepe",
"apellido": "martinez",
"Dni": 3912388
}

- Pedir los datos un usuario y crear la estructura arriba mencionada.
- Al finalizar la carga, agregar un nuevo campo “dniEsPar” (booleano) si el dni es par o no.
- Eliminar el campo nombre y apellido y crear un nuevo campo "nombreCompleto" por ejemplo: "nombreCompleto" : "pepe martinez".
- Convertir el json a cadena de texto (buscar el método, pista: "json to string").
